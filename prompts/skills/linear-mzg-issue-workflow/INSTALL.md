# Installing This Skill

This repo stores Codex skill docs under `prompts/skills/` so they can be versioned publicly.

To use it as an actual Codex skill, copy this folder into your Codex skills directory, for example:

- Windows: `%USERPROFILE%\\.codex\\skills\\linear-mzg-issue-workflow\\`

Then restart Codex (or reload skills) and invoke it by name: `linear-mzg-issue-workflow`.
